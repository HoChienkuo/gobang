# 简单五子棋实现

---

## 介绍

gobang 是一个可以和ai对战的五子棋程序，采用python实现，界面使用的是pygame(其中messagebox来自tkinter)
。五子棋UI界面部分参考的是知乎CMOS三好先生的文章(
【Python编程】pygame编写五子棋人机对战游戏)[https://zhuanlan.zhihu.com/p/24573632009]。
ai算法采用的是经过剪枝的minmax极大极小算法。<br/>

欢迎提出意见。<br/>

---

## 如何运行

```shell
python main.py
```
